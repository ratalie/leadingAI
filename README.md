# Leading in the AI Era - Presentation

Web-based presentation about AI leadership, resilience, and practical AI adoption.

## Topics Covered
- How AI is reshaping work and leadership
- Building a resilient career
- Using AI tools for better decisions
- Continuous learning and reinvention
- Responsible leadership in AI-driven environments
- Personal lessons and stories

## How to Run
1. Open `index.html` in your browser
2. Use arrow keys or space to navigate
3. Press 'F' for fullscreen, 'ESC' for overview

## Tech Stack
- Reveal.js for presentation framework
- HTML/CSS/JavaScript
